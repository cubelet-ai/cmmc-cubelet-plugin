Use the gap-analyst agent to generate consulting deliverables.

Available artifact types:
- gap-report: Full gap analysis report (DOCX)
- poam: Plan of Action & Milestones template (XLSX)
- evidence-guide: Evidence collection checklist by domain (DOCX)
- scorecard: Assessment readiness scorecard (PDF)
- all: Generate all four artifacts

Pull all necessary data from the CMMC Cubelet Platform, synthesize into
the requested format, and generate downloadable files.

Arguments: $ARGUMENTS
